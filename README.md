# EPOS - A Cozy Game About Culture & History

A pixel art adventure game exploring Kyrgyz history and culture, developed by Kurultai Startup.

## Project Overview

EPOS is a peaceful pixel art adventure game that takes players on a journey through Kyrgyz history and culture. Built with modern web technologies for a smooth, responsive experience.

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - Modern UI library
- **shadcn-ui** - Beautiful UI components
- **Tailwind CSS** - Utility-first CSS framework

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your system.

### Installation

```bash
# Clone the repository
git clone <YOUR_REPOSITORY_URL>

# Navigate to the project directory
cd epos_project

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
